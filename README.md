## 功能

>- 使用 `ctrl + win + 鼠标滚轮（按下）` 打开使用说明
>- 使用 `win + N` 打开写字板
>- 可以使用 `ttt` 来快速的输入时间。

以上是`temp.ahk`新的功能

>- 可以用`鼠标滚轮`和`键盘上的按键`组合来调节**音量**/**显示器的亮度**.
>- 快速切换歌曲(上一首,下一首)以及暂停。
>- 支持超多音乐软件
（只要这个音乐软件可以使用电脑（如果有）自带的音乐切换按键。）

- [ ] 会在每次启动脚本的时候弹窗提示键位.

*按键不可自定义*，~~除非你会ahk~~
